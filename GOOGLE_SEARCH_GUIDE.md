# Google检索方法完整指南

## 1. 基础搜索方法

### 1.1 单关键词搜索
```
医保改革
```
- 最简单的搜索方式
- 返回包含该词的所有结果
- 结果范围最广

### 1.2 多关键词搜索（空格分隔）
```
医保 改革 政策
```
- Google默认使用AND逻辑
- 返回同时包含所有词的结果
- 结果更精准

### 1.3 短语搜索（引号）
```
"医保改革政策"
```
- 精确匹配整个短语
- 词序必须一致
- 结果最精准但数量少

---

## 2. 高级搜索运算符

### 2.1 OR运算符（或）
```
医保改革 OR 医疗保障
```
- 返回包含任一关键词的结果
- 扩大搜索范围
- 适合同义词搜索

### 2.2 减号运算符（排除）
```
医保改革 -商业保险
```
- 排除包含特定词的结果
- 过滤无关内容
- 提高结果相关性

### 2.3 星号通配符（*）
```
医保*改革
```
- 匹配任意词
- 适合不确定的词组
- 发现新的表达方式

### 2.4 site:运算符（限定网站）
```
医保改革 site:gov.cn
```
- 只搜索特定网站
- 适合搜索官方信息
- 提高信息权威性

### 2.5 intitle:运算符（标题包含）
```
intitle:医保改革
```
- 只搜索标题包含关键词的页面
- 提高相关性
- 适合新闻搜索

### 2.6 inurl:运算符（URL包含）
```
inurl:news 医保改革
```
- 只搜索URL包含特定词的页面
- 适合特定类型内容
- 如新闻、博客等

---

## 3. 组合搜索技巧

### 3.1 AND组合（默认）
```
生物医药 产业 发展
```
- 所有词都必须出现
- 精准但可能遗漏相关内容
- **问题：** 过于严格，结果少

### 3.2 OR组合（扩展）
```
生物医药 OR 医药产业 OR 制药行业
```
- 任一词出现即可
- 覆盖面广
- **问题：** 可能包含不相关内容

### 3.3 混合组合（推荐）
```
(生物医药 OR 医药产业) 发展 政策
```
- 结合AND和OR
- 平衡精准度和覆盖面
- **最佳实践**

### 3.4 短语+关键词
```
"生物医药产业" 发展 政策
```
- 核心短语精确匹配
- 其他词灵活匹配
- 适合特定主题搜索

## 4. Google新闻特定技巧

### 4.1 时间限制
```
医保改革 after:2026-02-01
```
- 只显示指定日期后的结果
- 适合获取最新新闻

### 4.2 地区限制
```
医保改革 location:北京
```
- 限定特定地区的新闻
- 适合地方政策搜索

### 4.3 来源限制
```
医保改革 source:人民网
```
- 限定特定新闻源
- 提高信息可信度

```
医保改革
```

- 结果：5-10条
- 覆盖面广
- 相关性高

**✅ 推荐方案2：两词组合**
```
医保改革 政策
```
- 结果：3-5条
- 更精准
- 适合特定主题

**✅ 推荐方案3：OR组合**
```
医保改革 OR 医疗保障改革
```
- 结果：8-15条
- 覆盖同义词
- 不遗漏相关内容
