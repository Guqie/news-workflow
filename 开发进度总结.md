## 一、项目架构

### 1.1 目录结构
```
news-workflow/
├── references/
│   └── config.json              # 配置文件（关键词、来源）
├── scripts/
│   ├── google_news_crawler.py   # Google新闻爬虫 ✅
│   ├── rolling_news_crawler.py  # 滚动新闻爬虫 ✅
│   ├── searxng_crawler.py       # SearXNG爬虫 ✅
│   ├── json_to_csv.py           # JSON转CSV ✅
│   └── send_outlook_email.py    # 邮件发送 ⚠️
├── data/
│   └── raw/                     # 原始数据存储
└── 战略新兴产业关键词扩充.md    # 文档
```

### 1.2 技术栈
- **爬虫：** Python + GNews库
- **数据处理：** JSON → CSV
- **邮件发送：** SMTP（待完成OAuth 2.0）
