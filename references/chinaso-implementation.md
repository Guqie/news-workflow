# 基于中国搜索的新闻采集方案

## 一、中国搜索特点分析

### 1.1 优势
- ✅ 国内新闻覆盖全面
- ✅ 时间筛选功能强大（支持24小时内）
- ✅ 标题质量高（80%可通过标题初筛）
- ✅ 搜索结果相关性好

### 1.2 适用场景
- 教育人才板块新闻采集
- 医疗健康板块新闻采集
- 作为主要采集方式

---

## 二、实现方案

### 2.1 搜索URL构造

```python
base_url = "https://www.chinaso.com/search/pagesearch.htm"
params = {
    "q": "关键词",
    "time": "1",  # 1=24小时内
    "page": 1
}
```


### 2.2 关键词策略

#### 教育人才板块
**一级关键词（每日必搜）：**
- 教育政策
- 人才引进
- 高校改革
- 职业教育

**二级关键词（轮换搜索）：**
- 招生考试
- 就业市场
- 教育科技
- 人才培养


#### 医疗健康板块
**一级关键词（每日必搜）：**
- 医疗政策
- 医药监管
- 医院改革
- 生物医药

**二级关键词（轮换搜索）：**
- 医疗器械
- 健康产业
- 医保政策
- 药品审批


### 2.3 标题初筛规则（基于80%可筛选的特点）

#### 快速通过的标题特征
- ✅ 包含政策、发布、通知等关键词
- ✅ 包含具体机构名称
- ✅ 包含数据、报告等词汇
- ✅ 陈述性标题，非疑问句

#### 快速排除的标题特征
- ❌ 包含"震惊"、"必看"等标题党词汇
- ❌ 纯广告性质标题
- ❌ 娱乐八卦类标题
- ❌ 疑问句标题（大部分）


### 2.4 采集流程

```
1. 构造搜索URL（关键词 + 24小时限制）
   ↓
2. 获取搜索结果列表
   ↓
3. 标题初筛（80%在此完成）
   ↓
4. 提取通过的新闻详情
   ↓
5. 保存到raw/目录
```

### 2.5 每日采集计划

**教育人才板块：**
- 搜索4个一级关键词
- 每个关键词取前5条
- 标题初筛后预计保留12-15条
- 最终目标：6+条

**医疗健康板块：**
- 搜索4个一级关键词
- 每个关键词取前6条
- 标题初筛后预计保留15-20条
- 最终目标：8+条

## 三、优化建议

### 3.1 基于标题初筛的优化

**利用AI辅助标题筛选：**
```python
# 快速判断标题是否值得采集
def quick_title_filter(title):
    # 排除词
    exclude_words = ["震惊", "必看", "速看", "不看后悔"]
    if any(word in title for word in exclude_words):
        return False
    
    # 优先词
    priority_words = ["政策", "发布", "通知", "报告", "数据"]
    if any(word in title for word in priority_words):
        return True
    
    return True  # 默认通过
```

### 3.2 时间效率优化

- 标题初筛：2-3分钟
- 详情抓取：5-8分钟
- 总计：10分钟内完成采集
