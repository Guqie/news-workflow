# 新闻自动化工作流系统

自动化新闻采集、筛选、格式化工作流系统，专注于战略新兴产业和高科技产业领域。

## 📋 项目简介

将每日4-6小时的手动新闻检索工作缩短至30分钟，效率提升85-90%。

**覆盖领域：**
- 🚀 战略新兴产业（人工智能、人形机器人、低空经济、商业航天、量子通信、脑机接口、合成生物、可控核聚变等）
- 💻 高科技产业（芯片、半导体、算力、5G/6G、云计算、大数据、物联网、智能制造等）

## ✨ 核心功能

- ✅ 多源采集：Google新闻 + 滚动新闻 + 东方财富网等财经媒体
- ✅ 优化关键词：基于工作交接文档优化检索策略
- ✅ 智能筛选：7类信息分类标准（政策类、企业动态、技术突破等）
- ✅ 格式化输出：CSV格式，支持OneDrive自动上传
- ✅ 排除过滤：自动排除新能源汽车、风电、光伏等不相关领域

## 🚀 快速开始

```bash
# 1. 进入项目目录
cd news-workflow/scripts

# 2. 运行完整采集（战略新兴产业）
python3 news_aggregator.py --sector strategic_emerging --hours 24

# 3. 运行完整采集（高科技产业）
python3 news_aggregator.py --sector hightech --hours 24

# 4. 快速测试优化后的配置
python3 test_optimized_config.py

# 5. JSON转CSV（如需要）
python3 convert_json_to_csv.py <输入JSON> <输出CSV>
```
